# Sample Markdown

A sample document to exercise the **markdown** preview renderer.

## Features

- Bullet lists
- `inline code`
- [links](https://github.com)

## Table

| Renderer | Mode        |
| -------- | ----------- |
| code     | full / diff |
| markdown | preview     |
| json     | tree        |
